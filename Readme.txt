This is the xelatex compiled pdf for English Translations of Quran
written by

1)maududi
2)pikthall
3)Sahih international

The text form of translations are downloaded from [www.zekr.org]


gentextforquran.py is to be run to generate the tex files


xelatex maududi.tex 

gives maududi.pdf which is maudoodi's quran translation

xelatex pickthall.tex 

gives pickthall.pdf which is pickthall's quran translation

xelatex sahih.tex 

gives sahih.pdf which is sahih international's quran translation
